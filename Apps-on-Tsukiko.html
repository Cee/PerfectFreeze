Apps on Tsukiko

By Cee on Aug 19, 2015

Q：「Tsukiko / 月子」是什麼？

A：僅僅是我的 Macbook 的名字。

（無語…）

今天正好整理了一下我的應用程序，順便推薦了一些給了 Lynn。

以下均為一句話概括！

  1. 1Password 5：強大的密碼記錄和生成的軟件！

  2. Aegisub：字幕組打軸特效專用。

  3. Alfred 2：快速啟動神器，配合 Workflow 使用更佳。

  4. AliWangWang：買買買，剁手必備！

  5. Alternote：好用的 Evernote 客戶端，可惜略貴。一直在蹭 Beta 版的。

  6. Amphetamine：防止電腦休眠，自己也不常用。

  7. Android File Transfer：管理安卓手機文件的工具。

  8. Android Studio：Google 推出的基於 IntelliJ IDEA 的安卓開發 IDE，比 Eclipse 不知道高到哪裡去了。配合 Genymotion 使用更好！

  9. AppCode：JetBrains 家用來寫 Objective-C 的 IDE。

  10. Atom：文本編輯器，卡卡卡，畢竟是 Web，自己還是喜歡用 Sublime。

  11. Bartender：整理 Menubar 專用。喜歡簡潔乾淨的 Menubar 必備！

  12. BetterTouchTool：手勢還有控制屏幕大小。（有部份 Moom 的功能，El Capitan 要出了還是不買了）

  13. BitTorrent Sync：分佈式同步工具，方便自己在不同電腦上同步文（tu）件（pian）。

  14. Blackmagic Disk Speed Test：磁盤測速，每年也就用那麼兩次。

  15. Brackets：很不錯的做網頁的 IDE，被 Adobe 收購了。

  16. Charles：網絡抓包工具，很實用！

  17. CleanMyMac 2：買不起 3，繼續用 2 來清理我的電腦。

  18. CodeRunner：用過試用版沒有買，很棒的小巧的運行代碼的編輯器。

  19. Dash：查文檔專用！不過 10 秒的等待很坑爹。

  20. Day One：日記，偶爾做 Markdown 編輯器用。

  21. Deploymate：檢測代碼中是否有被廢棄或者無法在低版本上運行的 API。

  22. Dropbox：同步，容量好小 QwQ。

  23. Droplr：分享短連接，可以分享圖片、視頻和文字或者其他文件。

  24. Duet：讓你的 iPhone / iPad 成為第二塊顯示屏！

  25. Evernote：臃腫的大象，偶爾用了。

  26. Fantasitcal 2：日曆管理。

  27. FileZilla：FTP GUI 工具。

  28. FirefoxDeveloperEdition：你是個前端都會備著這個。

  29. GenyMotion：配合 AS 使用的，作為 Android 虛擬機。

  30. GhostNote：丟過數據，不解釋。

  31. GoAgentX：咳咳。

  32. Google Chrome：吃我內存啦 (╯‵□′)╯︵┻━┻。

  33. Hammer：構建網頁。

  34. HandBrake：壓制視頻。

  35. Haroopad：用到現在感覺最好而且是免費的用來寫 Markdown 文件的 App！

  36. HockeyApp：收集 Crash 信息。

  37. IconJar：存儲 icon，分類。

  38. ImageOptim：壓縮圖片。

  39. Inboard：Xhacker 寫的一個用於素材歸檔的 App，很好用。

  40. IntelliJ IDEA 14：JetBrains 家用來寫 Java 的 IDE。

  41. iStat Menus：放在 MenuBar 上用於顯示系統信息，還能控制風扇轉速！

  42. iTerm：默認的 Terminal，網上很很多文章介紹為什麼要選擇 iTerm。順便使用 zsh 效果更佳。

  43. Karabiner：當接上外接鍵盤時屏蔽內置鍵盤。

  44. Keka：解壓，想換成 Entropy 了（沒錢啊）！

  45. Kindle：看電子書。

  46. Knock：利用 iPhone 敲兩下來解鎖電腦。

  47. Koala：為 Sass / Less 生成 Css 文件。

  48. LineIn：混音專用，配合 Sunflower。

  49. MailPlane 3：收取 Gmail 郵件，我的默認郵件客戶端。

  50. Manico：快速切換 App。

  51. Memory Clean：清理內存。

  52. Miao：刷新浪微博專用，以前是要付費的，現在因為新浪微博 API 的限制改為免費。

  53. Microsoft OneNote：替代了 Evernote。

  54. Microsoft Remote Desktop：用於連接我的 Windows 電腦。

  55. Mounty：掛載 NTFS。

  56. Movist：最好用的視頻播放器！

  57. MPlayerX：第二好用的視頻播放器！

  58. NeteaseMusic：網易云音樂，偶爾用，主要還是 iTunes 和 Vox。

  59. OBS：Twitch 直播。

  60. Parallels Desktop 9：奸商，買不起。

  61. Paste：剪貼板，很贊（可以用 Alfred 自帶的，不過沒這個好）。

  62. Paw：超棒的調試 API 的工具。

  63. Phpstorm：JetBrains 家用來寫 PHP（世界上最好的語言）的 IDE。

  64. Pixate Studio：做交互稿用。

  65. Pocket：稍後閱讀！

  66. PopClip：選中文字后有個小的彈窗，能做很多事情！

  67. Prepo：裁剪 icon 圖標。

  68. Pushbullet：給其他設備發送通知。

  69. QQ：聊天。

  70. Quartz Composer：做原型和交互。

  71. Quip：同步一些文檔，也沒怎麼用過。

  72. RescueTime：記錄我一天都幹了什麼！配合 WakaTime 使用！

  73. Reveal：iOS 開發用，強大的透視圖。

  74. RightFont：挑選一款字體真的很難，它能幫到你。

  75. Scroll Reverser：鼠標滾輪能夠正常滾動。

  76. Server：自帶一個 Web 服務器還有 Xcode Server。

  77. Sip：吸取顏色，自動變成代碼。

  78. Skala Preview：同步到手機上預覽圖片（設計稿）。

  79. Sketch：替代了 PS 和 AI 做設計稿了，矢量的哦！

  80. Slack：團隊交流工具。

  81. SoundFlower：錄音混音工具。

  82. SourceTree：GUI 的 Git 管理工具。

  83. Steam：買，買，買，男人的淘寶。

  84. Sublime Text：最好的文本編輯器。

  85. TeamViewer：遠程桌面軟件。

  86. Telegram：最好用的 IM，多平台支持。

  87. Tunnelblick：OpenVPN。

  88. Tweetbot：替代 Twitter 的 Twitter 客戶端。

  89. Twitter：被 TweetBot 替代啦！

  90. Typed：Markdown 編輯器。

  91. Typora：又是一個 Markdown 編輯器。

  92. VirtualBox：裝 GenyMotion 必需。

  93. VOX：音樂播放器，支持 Flac 等 iTunes 不支持的格式。

  94. WebStorm：JetBrains 家用來寫 Web 的 IDE。

  95. WeChat：微信 Mac 客戶端。

  96. WWDC：學習 WWDC 中的 Session，在 GitHub 上開源。

  97. Xcode：碼農必備。

  98. Yu：白噪音軟件。
